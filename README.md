# 24hwmn
An interactive exhibition website for “24 Hours of a Woman’s Life” (10k+ visitors), built using PHP, JavaScript, HTML/CSS, and MySQL.
The website is designed for use at the exit of the exhibition, allowing visitors to leave a message in a digital guestbook, share the exhibition on social media, and discover local organizations supporting women’s rights. After completing these three actions, users receive a personalized message from the character they portrayed.

The website of the exhibition is : https://www.24h-wmn.org/
